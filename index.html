GitHub fue fundado en 2008 como una plataforma de alojamiento de repositorios de código Git. Ofrece herramientas para el control de versiones, seguimiento de problemas y colaboración en proyectos de software. Rápidamente se convirtió en un lugar central para desarrolladores de todo el mundo, tanto para proyectos de código abierto como comerciales. En 2018, Microsoft adquirió GitHub por $7.5 mil millones, pero sigue operando de forma independiente. Hoy en día, GitHub es la plataforma líder para el desarrollo colaborativo de software.
